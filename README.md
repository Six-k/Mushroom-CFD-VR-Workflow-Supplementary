# Mushroom CFD-VR Workflow — Supplementary Materials

Supplementary materials for the manuscript:

**"Development of an Open-Source Integrated Workflow for Airflow Simulation and Virtual Reality Visualisation in Mushroom Cultivation Facilities"**

---

## Contents

### airFields_V2

Blender add-on for importing OpenFOAM flow field results into Blender. The intermediate file format is **X3D**.

### airFields_V3

Blender add-on for importing OpenFOAM flow field results into Blender. The intermediate format uses **VTK/VTP** files.

### viewport_vr_preview_v2

A fork of Blender's built-in VR Scene Inspection add-on. In VR view, it provides a panel for toggling object visibility.

---

## Installation

1. Copy the desired add-on folder (e.g., `airFields_V3`) into Blender's add-ons directory, or use **Edit > Preferences > Add-ons > Install...** in Blender.
2. Enable the add-on.
